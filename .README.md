# Golf Club Golden Elephant — WordPress Theme

- This project is a part of a technical task. A custom theme for the Golden Elephant golf club.

## What you'll need
- WordPress 6.x
- Advanced Custom Fields (ACF) plugin

## Getting started
1. Drop the `golf-club-golden-elephant-theme` folder into `/wp-content/themes/`
2. Activate it under Appearance -> Themes
3. Install and activate the ACF plugin
4. Hit Save on Settings -> Permalinks

## Importing the content

**XML**
1. Go to Tools → Import → WordPress
2. Upload `demo-content.xml`
3. Once imported, go to Settings → Reading and set the homepage to the imported Home page

Note: tournament images are not included in the XML export. I have made a placeholder tournament image for preview on hover. 
After importing, add images to past tournaments via WP Admin → Tournaments.

**SQL**
1. Import `demo-content.sql` via Adminer, phpMyAdmin or LocalWP's database tool
2. The URLs are set to `golf-club-golden-elephant.local` so if needed, update that under Settings → General

## Editing the content

Most of the page content (hero title, subtitle, background image, about section text and footer content) 
can be edited from **Pages → Home → Edit**
There are three tabs - Hero, About, Footer

The logo and navigation menu are managed separately from **Appearance → Customize**.
